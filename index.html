Better Simple Auto Dark Mode

A Chrome extension that detects whether a site is already dark, and only switches the ones that aren't.

View on GitHub Privacy Policy

What it does

  • Auto-detects dark sites by sampling background & text luminance — leaves them alone.
  • Hints color-scheme: dark first so sites with their own dark theme switch natively.
  • Falls back to generated dark CSS that remaps light surfaces, text, borders, and controls.
  • Leaves media colors natural instead of flipping the entire page.
  • Slider tunes the generated palette from soft gray to near-black.
  • Per-site whitelist with subdomain matching.
